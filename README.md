## Setup

```sh
$ git clone https://github.com/llyt/frontend-project-lvl1.git
```
```sh
$ make install
```
```sh
$ make publish
```
```sh
$ npm link
```
## Even

Answer "yes" if number even otherwise answer "no".

```sh
$ brain-even
```
